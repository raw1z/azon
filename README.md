Azon
====

Todos made fun for touch typists and vim's fans.

Requirements
============

* platform: node.js
* language: coffee-script
* database: mongodb

Installation
============

1. git clone https://github.com/raw1z/azon.git
2. cd azon && npm install
3. coffe app.coffee
